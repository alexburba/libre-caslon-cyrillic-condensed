# Libre Caslon Cyrillic (Text Regular & Condensed Regular)

![Screenshot 2025-11-17 at 02 29 18](https://github.com/user-attachments/assets/8abf6123-7bf6-4e7a-8ebc-8337abc5fdaa)

_Eng:_
Cyrillic versions of the [Libre Caslon Text Regular](https://github.com/impallari/Libre-Caslon-Text) and [Libre Caslon Condensed Regular](https://github.com/ertekinno/libre-caslon-condensed). Just a 4 days and I have managed to add cyrillic glyphs for RU, BY, UA. Didn't bother with a proper kerning — just fixed obvious pairs along the way.

_Ru:_
Кириллические версии шрифтов [Libre Caslon Text Regular](https://github.com/impallari/Libre-Caslon-Text) и [Libre Caslon Condensed Regular](https://github.com/ertekinno/libre-caslon-condensed). Все сделал за 4 дня. Кернинг фиксил только там, где бросалось в глаза.

- **Scripts:** Latin, Cyrillic
- **Сyrillic:** RU, BY, UA
- **Styles:** Regular
- **Kerning:** 5%
- **Plans**: a proper kerning for all the cyrillics someday
- **License:** SIL OFL 1.1
